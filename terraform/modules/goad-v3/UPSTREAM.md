# GOADv3 attribution and adaptation boundary

This module's five-host naming and lab topology are adapted for OCI from
[Orange-Cyberdefense/GOAD](https://github.com/Orange-Cyberdefense/GOAD), pinned
for the M11 design review at commit `992307adf944b934a3b76a2f56a637104c54b805`.

GOAD is licensed under GPL-3.0. The upstream project remains the authoritative
source for its vulnerable Active Directory provisioning logic and license.
This repository does not copy credentials or topology from an existing GOAD
deployment. OCI instances are private and bootstrap through OCI Agent Run
Command.
